## About The Project

Development of Bash text utilities: cat, grep.
Read 'src/Task.md' to get closer look at the project.

## Installation

```sh
    git clone git@github.com:umed-hotamov/school21.git
```

## Running The Tests

To run cat tests:

```sh
cd cat
make
```

To run grep tests:

```sh
cd grep
make
```

## Usage

Cat:

```sh
s21_cat [OPTION]... [FILE]...
```

Grep:

```sh
grep [OPTION...] PATTERNS [FILE...]
```
